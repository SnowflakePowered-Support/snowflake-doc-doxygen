Detailed Description

Represents a store of controller mapping profiles IMappedControllerElementCollection

Mapping profiles are stored in standard Stone controller format in a database, keyed on the Stone layout name of the emulated controller, the device name of the enumerated real controller Input.Device.IInputDevice, and a profile name, user specified and 'default'.

The IMappedControllerElementCollectionStore does not consider which player and only stores a list of profiles for each combination of real to emulated controller.
