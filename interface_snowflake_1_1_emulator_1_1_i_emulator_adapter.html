Detailed Description

Emulator adapters manage the instance generation of a certain emulator that supports one or more Stone ROM formats. They are responsible for creating an IEmulatorInstance that will generate correect configuration given the available ports and input mappings, as well as responsible for supplying valid configuration for a given game.

IEmulatorInstance Snowflake.Emulator.IEmulatorAdapter.Instantiate ( IGameRecord  gameRecord,
IFileRecord  romFile,
int  saveSlot,
IList< IEmulatedPort >  ports 
)

Creates an IEmulatorInstance that can be used to spawn the emulator process and generate valid cofiguration. Instances are trusted to generate valid configuration in accordance with the given parameters, including providing the correct save file and bios directories.

However, it is NOT the responsibility of the adapter to launch the game.

Parameters
gameRecordThe IGameRecord associated with this instance
romFileThe IFileRecord belonging to the game, that is the entry point for the emulation
saveSlotThe save manager slot in which this instances save files are stored
portsThe list of emulated controler ports active for this instance.
Returns
The prepared emulator instance.
