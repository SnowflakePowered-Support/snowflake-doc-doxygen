Detailed Description

Represents a provider for queried data requiring a search query and a platform.

A query provider should only implement QueryBestMatch(string, string) and Query(string, string) Other types of queries are implemented as functions that take in a single IMetadataCollection, and be marked with a ProviderAttribute. Functions that require a certain metadata to be available should mark such metadata as RequiredMetadataAttribute
